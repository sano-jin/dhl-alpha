# dhl-alpha
_Experimental_

![logo](./docs/DHL-logo-alpha.svg)

A minimal compiler and runtime for a language based on graph rewriting.

## Getting Started
### Prerequisites
- [opam](https://opam.ocaml.org/)

### Installation
```bash
git clone https://github.com/sano-jin/dhl-alpha
cd dhl-alpha
opam install .
dune build
```

## Usage

```bash
./dhl example/append.dhl -t
```
