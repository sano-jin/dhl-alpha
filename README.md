# dhl-in-ocaml

A minimal compiler and runtime for a language based on graph rewriting.

## Getting Started
### Prerequisites
- [opam](https://opam.ocaml.org/)

### Installation
```bash
git clone https://github.com/sano-jin/dhl-in-ocaml
cd dhl-in-ocaml
opam install .
dune build
```

## Usage

```bash
./dhl example/append.dhl -t
```
